# BusMarg
Bus tracking application with real-time location tracking, driver management, and Firebase authentication

# Features

1) For Passengers:
a) Live bus tracking on interactive maps
b) Search and filter buses
c) Follow and like favorite buses
d) View driver details and bus information

2) For Drivers:
a) Complete driver profile setup
b) Upload profile picture
c) Go live with real-time location tracking
d) Manage online/offline status

# Authentication:

Google OAuth sign-in
Email/Password registration and login
Secure Firebase authentication

# Tech Stack

HTML5, CSS3, JavaScript
Firebase (Authentication, Database, Storage)
Geolocation API
Font Awesome Icons
Google Fonts (Inter)

# Getting Started

Clone or download the project
Open index.html in your browser
Configure Firebase credentials in the app
Sign in with Google or Email
Start tracking or register as a driver

# Setup

Create a Firebase project at firebase.google.com
Enable Authentication (Email/Password & Google OAuth)
Enable Realtime Database and Cloud Storage
Add your Firebase config to the application
